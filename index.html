    Automatic Emotion Detection in Images Data


    Tools:
    • Python 3.6 (numpy, tensorflow, keras, OpenCV, dlib, skimage)


    Applied CNN and HOG+SVM in detecting human emotions and compared the accuracy.

    By using CNN, the resulting model with two convolutional layers and full landmarks were processed in keras by inputting images into the convolutional neural network and guiding CNN by normalized landmarks.

    By using HOG+SVM, detected facial features by extracting the histogram of oriented gradients feature descriptors and classified by using support vector machine.

    Document Word Counter


    Tools:
    • Python 3.6
    • Anaconda
    • Apache Spark 2.2.1
    • Pyspark 2.2.1


    Implemented a basic word counter in Apache Spark and resulted in a customized word-count dictionary.

    According to the books inputted, the word counter will detect the words, record the corresponding counts, elminate the stopwords, filter out the punctuations, and provide the top n most frequent words in a dictionary. Otherwise, depends on the TF-IDF of each specific word, the counter will also produce the top t words with highest TF-IDF scores.

    Scalable Document Classification


    Tools:
    • Python 3.6
    • Anaconda
    • Apache Spark 2.2.1
    • Pyspark 2.2.1
    • Google Cloud Platform


    Implemented Naive Bayes classifier on Apache Spark and resulted in 94.52% classification accuracy in large dataset.

    The project uses Reuters Corpus but only specifies four categories that end with CAT. We applied several preprocessing skills like removing punctuations, words stemming, and removing stopwords. The Naive Bayes classifier is also implemented with Laplace smoothing to avoid zero count problems.
